Virtual Environment

Always work inside a virtual environment. Label Studio pulls in dozens of packages — installing globally will conflict with ultralytics and opencv.

⭐
Activate the venv every time you open a new terminal — it does not persist between sessions.
🐧 Pi / macOS / Linux
🪟 Windows
bash 
# Create once
python -m venv aeroclean_env

# Activate every session
source aeroclean_env/bin/activate

# Prompt becomes: (aeroclean_env) pi@raspberrypi:~$
# Create once
python -m venv aeroclean_env

# Activate every session
aeroclean_env\Scripts\Activate.ps1
Deactivate when done
bash
deactivate
✅
aeroclean_env/ is already in .gitignore — it will never be committed.

Tips for a good dataset

💡

Vary lighting

Overhead fluorescent, natural window, lamp at an angle — the Pi will see all of these.

🖊️

Vary board coverage

Lightly smudged, heavily marked, ghost marks, partially erased — capture the spectrum.

📐

Vary angle slightly

Straight-on plus small tilts (±15°). Match the mounting height you'll use in production.

Steps

1. Click + Create Project — give it a name, e.g. AeroClean
2. Go to Labeling Setup → choose Object Detection with Bounding Boxes
3. Add labels clean_board and dirty_board — order matters (class IDs 0 and 1)
4. Click Import and drag your images/raw/ folder to upload all images
ℹ️
Import both clean_board/ and dirty_board/ subfolders at once. Label Studio shows them all in a single task list.

Label Studio controls

ActionHow
Select labelClick clean_board or dirty_board in the left panel, or press 1 / 2
Draw boxClick and drag to draw a rectangle around the board surface
ResizeClick box → drag corners or edges
Delete boxClick box → Backspace
Submit & nextClick Submit bottom-right, or Ctrl + Enter
Skip imageSkip for blurry or unusable frames
🎯
One box per image. Draw a single tight box around the full board surface. Consistency beats perfection.
⚠️
Decide a rule for ambiguous boards (e.g. "any visible residue = dirty") and stick to it. Inconsistent labels are the #1 cause of poor mAP.

Export & Dataset Structure

Export from Label Studio

1. Project dashboard → click ⋯ → Export
2. Select YOLO format
3. Download the zip → save to project root
4. Extract into images/ and labels/

YOLO label format

yolo .txt
# class  x_center  y_center  width  height  (0–1 normalised)
1 0.512 0.498 0.883 0.762

Zip for Colab

bash
zip -r aeroclean_dataset.zip \
  images/ labels/ train/dataset.yaml
